# doc-flow PPT 设计方案

## 项目概述

为 doc-flow 项目创建一个简洁高级的网页版 PPT 演示文稿，用于项目介绍和展示。

## 设计目标

1. **风格一致性**：与 LangGraph 官网视觉风格保持一致
2. **内容精炼**：控制在 10 页以内，高度概括核心要点
3. **技术展示**：清晰展示项目的技术价值和创新点
4. **视觉简洁**：极简科技风，大量留白，突出重点

## 技术方案

- **框架选择**：Reveal.js
- **展示形式**：分页切换式传统 PPT
- **主题风格**：浅色主题，LangGraph 蓝色调
- **内容呈现**：要点列举配合图标

## 视觉规范

### 色彩方案
- **主色调**：LangGraph 蓝 (#0066FF)
- **背景色**：纯白 (#FFFFFF)
- **主体文字**：深灰 (#333333)
- **辅助文字**：浅灰 (#666666)

### 字体规范
- **英文字体**：Inter
- **中文字体**：PingFang SC
- **标题字号**：48-64px
- **正文字号**：24-32px

### 排版原则
- **留白比例**：内容区域占页面 60-70%
- **对齐方式**：居中对齐为主
- **层级关系**：通过字号、颜色深浅区分
- **图标使用**：简洁的线性图标，与文字搭配

### 动效设计
- **页面切换**：淡入淡出效果
- **过渡时间**：300-500ms
- **交互反馈**：按钮悬停变色

1. **环境搭建**

2. **页面开发**

3. **样式定制**

4. **部署上线**
   - 构建静态文件

   - 生成分享链接
